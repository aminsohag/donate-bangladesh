Important FAQ

What is the Document Object Model(DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a web page as a tree of objects, with each object corresponding to a part of the page, such as elements, attributes, and text

How do you select an element from DOM?

In JavaScript, you can select elements from the DOM (Document Object Model) using various methods, depending on the criteria (such as id, class, or tag name) and the number of elements you want to select. Below are the most commonly used methods to select elements from the DOM: 1. getElementById(), 2. getElementsByClassName(), 3. getElementsByTagName(), 4. querySelector(), 5. querySelectorAll()

What is event delegation in the context of the DOM, and why it is useful?

Event delegation is a technique in JavaScript where a single event listener is attached to a parent element instead of multiple individual event listeners attached to each child element. The parent element listens for events on its children, and when an event occurs on a child, the event "bubbles" up to the parent, allowing the parent to handle the event.

How do you manipulate an elements attributes and styles using the DOM?

DOM Manipulation (Attributes & Styles): Attributes: Get Attribute: element.getAttribute('attr') Set Attribute: element.setAttribute('attr', value) Remove Attribute: element.removeAttribute('attr') Styles: Get Style (Computed): window.getComputedStyle(element).property Set Style: element.style.property = 'value' Remove Style: element.style.property = '' Classes: Add Class: element.classList.add('class') Remove Class: element.classList.remove('class') Toggle Class: element.classList.toggle('class') Check Class: element.classList.contains('class')
